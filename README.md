# iyam_ui_kit

`iyam_ui_kit` adalah **Flutter UI Kit & Design System** yang **customizable, theme-driven, dan reusable**
untuk digunakan di berbagai aplikasi.

Package ini dirancang untuk:

- mengurangi duplikasi UI
- menjaga konsistensi desain
- mudah dikustom dari aplikasi
